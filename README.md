O código apresentado neste trabalho tem como finalidade implementar técnicas aprendidas na disciplina relacionadas a programação orientada a objeto e interface gráfica (GUI), utilizando a linguagem de programação “Python” e técnicas vinculadas a biblioteca do Tkinter para a formação da GUI. O conteúdo tem como eixo principal oferecer maneiras automatizadas e mais simplificadas de realizar cálculos relacionados a bioquímica e biotecnologia, sendo elas: cálculo de molaridade de uma solução, cálculo de diluição de uma solução estoque com molaridade conhecida e uma calculadora simples que pode ser usada para relações comuns no dia a dia dentro da rotina de laboratório, além de oferecer uma interface divertida, amigável e intuitiva que torna o trabalho como algo mais delicado e leve. O público-alvo é principalmente estudantes e pesquisadores que estudam e trabalham em áreas relacionadas a bioquímica e química de soluções.
